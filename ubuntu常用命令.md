# ubuntu常用命令





```bash
apt update      				#查询软件包更新（不执行更新操作）
apt upgrade     				#执行软件包更新
passwd    						#重置当前用户密码（可以跟用户名，重置指定用户密码）
ufw	status						#查看防火墙状态  inacvice（不活动） acvice（活动）
ufw allow 22    				#防火墙放行指定端口
ufw delete allow 22 			#防火墙关闭指定端口
ufw enable						#开启防火墙
ufw disable						#关闭防火墙
ufw reload						#重启防火墙（修改防火墙配置后需重启生效）

```
